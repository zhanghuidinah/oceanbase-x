Quick Start

Deploy PowerRAG in 5 Minutes

Shell — Docker Compose Launch
# Clone PowerRAG repository
git clone https://github.com/oceanbase/powerrag.git
cd powerrag

# Docker Compose launch (built-in OceanBase)
docker compose up -d

# Or pull directly from Docker Hub
docker pull oceanbase/powerrag:latest
Configure LLM Models
# Configure models in PowerRAG console
# Settings → Model Providers → Add Ollama / OpenAI / vLLM

# Recommended: Ollama local deployment
ollama pull llama3.2    # Chat model
ollama pull bge-m3      # Embedding model

# PowerRAG Settings
Chat Model: llama3.2
Embedding Model: bge-m3
Base URL: http://host.docker.internal:11434
Create Knowledge Base & Start Chat
1. Knowledge Base → New → Upload documents (PDF/Markdown/Excel)
2. Select chunking strategy → Auto parse & embed
3. Chat → Link knowledge base → Start RAG conversation
4. Agent → Create Agent Flow → Add Memory node
Three steps: 1) Docker launch 2) Configure LLM models 3) Create knowledge base and start chatting. Supports Ollama local models.
