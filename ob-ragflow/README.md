# OceanBase × RAGFlow

PowerRAG：多模态检索增强生成 — 基于 OceanBase 的企业级 RAG 解决方案。

## 内容概要

1. **RAGFlow 简介** — 深度文档解析、可组分块、Agent 编排、Memory 管理
2. **PowerRAG 介绍** — 基于 RAGFlow + OceanBase 的企业级增强版
3. **技术架构** — PowerRAG 扩展层 → RAGFlow 核心 → OceanBase 底座
4. **核心能力** — Memory 管理、Agent Chat、知识库治理、多管理员支持
5. **OceanBase 替代 MySQL** — 向量检索、全文检索、ACID 一致性、运维简化
6. **PowerMem Memory 系统** — 记忆提取、持久化、智能检索、LOCOMO SOTA
7. **企业级实践要点** — 数据安全、可观测性、高可用、弹性扩展
8. **5 分钟部署** — Docker Compose 启动、LLM 配置、知识库创建
